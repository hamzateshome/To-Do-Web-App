# 📝 Text Automation Tool

A simple web-based Text Automation Tool built using **HTML, CSS, and JavaScript**.  
It helps users quickly manipulate text with useful formatting utilities in real time.

---

## 🚀 Features

- Convert text to **UPPERCASE**
- Convert text to **lowercase**
- Remove **extra spaces**
- Count **words and characters**
- Clear all text instantly
- Change **font style**
- Real-time output updates

---
## 📌 How to Use

1. Open the project by running `index.html` in any browser  
2. Paste or type your text inside the input box  
3. Choose any action button:
   - **UPPERCASE** → Converts all text to capital letters  
   - **lowercase** → Converts all text to small letters  
   - **Remove Spaces** → Cleans extra spaces in text  
   - **Count** → Shows number of words and characters  
   - **Clear** → Deletes all text  
   - **Font Change** → Switches font style  
4. View results instantly on the same page — no reload needed
